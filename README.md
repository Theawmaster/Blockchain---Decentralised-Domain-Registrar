# Decentralized Domain Registry

A blockchain-based decentralized domain registrar built with [Hardhat](https://hardhat.org/) and Solidity.  
The project allows users to register, manage, and transfer domain names on-chain securely.

## Features (Planned / Implemented)
- Domain registration with unique names
- Ownership transfer between users
- Domain resolution (map domain → wallet address / metadata)
- Transparent, immutable records stored on blockchain

## Tech Stack
- **Smart Contracts**: Solidity  
- **Framework**: Hardhat (TypeScript)  
- **Testing**: Mocha + Chai  
- **Dependencies**: OpenZeppelin libraries  
- **Frontend (optional)**: React / Next.js  

---

## Installation & Setup
```bash
# clone the repo
git clone https://github.com/<your-username>/decentralized-domain-registry.git

# install dependencies
cd decentralized-domain-registry
npm install

# compile contracts
npx hardhat compile

# run tests
npx hardhat test
```
